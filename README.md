# CSharp-Programming-Jorney
## Core Fundamentals & Logic (C#)
In questa sezione ho raccolto le basi della programmazione procedurale e orientata agli oggetti, affrontando la gestione del flusso e le strutture dati fondamentali.

## [Esercizi Cicli For.cs]: Strutture Iterative e Array
**Implementazione:** Implementazione di cicli (for) e (foreach) per la manipolazione di array di stringhe e numerici.
**Sviluppo:** Sviluppo di algoritmi per calcoli matematici (tabelline, somme cumulative, quadrati.
**Gestione:** Gestione di liste di dati (elengo invitati, calcolo spesa totale).

## [Esercizi Cicli While.cs]: Logica Condizionale e Controllo Input
**Implementazione:** Implementazione di cicli (while) e (do-while) per la validazione robusta dell'input utente (es.controllo temperatura reattore, accesso tramite password).
**Sviluppo:** Sviluppo di sistemi interattivi comolessi (simulazione di Bancomat, distributore di bevande, gioco "Indovina il numero").

## [Metodi e array.cs]: Metodi, Overloading e Matrici
**Modularità:** Creazione di metodi riutilizzabili con signature multiple (Method Overloading) per calcoli geometrici e sistemi di saluto dinamici.
**Strutture Dati Avanzate:** Gestione di madrici bidimensionali e tridimensionali per modellare scenari reali come la gestione di magazzini e la mappatura dei posti di un cinema.
**Algoritmi di Ricerca:** Implementazione di cicli annidati per trovare valori minimi/massimi e calcolare medie su dataset simulati.

## Tecnologie utilizzate
**Linguaggio:** C#.
**Ambiente di lavoro:** Visual Studio.
